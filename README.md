# Market Deck

A local dashboard for ranking financial assets (ETFs, stocks, commodities) by price momentum and visualizing monthly returns in a heatmap. Supports multiple watchlists, dynamic currency conversion, and live editing via a built-in editor panel.

---

## Features

- **Momentum Rankings** — Rank assets by 1M / 3M / 6M / 12M trailing returns, identify the top N to hold equally weighted
- **Monthly Heatmap** — 12-month grid of individual monthly returns with a trailing 12M total
- **Multi-List Support** — Sidebar with multiple watchlists (US Sectors, EU Sectors, Country ETFs, Commodities, Global 99)
- **Dynamic Currency Conversion** — All prices converted to a global base currency (configurable) using live historical FX rates from Yahoo Finance
- **Built-in Editor** — Add/edit/delete watchlists and tickers, change tags and currencies, update the global base currency — all persisted to SQLite
- **Live Data** — Prices fetched in real-time from Yahoo Finance via CORS proxies

## Getting Started

### Prerequisites

- **Python 3.10+** (tested on 3.12)
- **pip packages**: `fastapi`, `uvicorn`, `pydantic`, `yfinance`

### Installation

```bash
pip install fastapi uvicorn pydantic yfinance
```

### First-Time Setup (Database Migration)

If `data.db` does not exist yet, generate it from the legacy JSON files:

```bash
python scripts/migrate.py
```

This reads `data/lists.json` and `data/colors.json` and creates `data.db` with all watchlists, tickers, tag colors, and default settings (base currency = EUR).

> **Warning:** Running `migrate.py` again will **delete and recreate** `data.db`, wiping any edits made through the UI.

### Running the Server

```bash
python server.py
```

Open **http://localhost:8000** in your browser.

## API Endpoints

All endpoints are served by `server.py` on port 8000.

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/init` | Bootstrap payload: settings, tag colors, and all watchlists with tickers |
| `POST` | `/api/prices` | Batch fetch historical prices via yfinance. Body: `{"tickers": ["AAPL", ...]}` |
| `GET` | `/api/settings` | Get all settings |
| `PUT` | `/api/settings/{key}` | Update a setting (e.g. `GLOBAL_BASE_CURRENCY`) |
| `POST` | `/api/lists` | Create a new watchlist |
| `PUT` | `/api/lists/{slug}` | Update watchlist metadata |
| `DELETE` | `/api/lists/{slug}` | Delete a watchlist and all its tickers |
| `POST` | `/api/lists/{slug}/tickers` | Add a ticker to a watchlist |
| `PUT` | `/api/tickers/{id}` | Update a ticker (symbol, name, tag, currency) |
| `DELETE` | `/api/tickers/{id}` | Remove a ticker |
| `PUT` | `/api/tag-colors/{tag}` | Update a tag's color scheme |

## Database Schema

**`settings`** — Key-value store (e.g. `GLOBAL_BASE_CURRENCY` → `EUR`)

**`watchlists`** — `id`, `slug`, `name`, `short_name`, `category`, `description`, `tag`, `currency`

**`tickers`** — `id`, `watchlist_id` (FK), `symbol`, `name`, `tag`, `currency`, `sort_order`

**`tag_colors`** — `tag` (PK), `bg`, `text`, `border`

---

## Currency Conversion

- The app converts all asset prices to the **global base currency** (set in Settings, default: `EUR`)
- FX rates are fetched from Yahoo Finance (e.g. `EURUSD=X`, `JPYEUR=X`)
- Special cases:
  - **`USX`** (US Cents) — divided by 100, no FX fetch needed
  - **`GBp`** (British Pence) — uses `GBPXXX=X` rate, then divides by 100

## Editor Usage

1. Click the **✎ Edit** button in the page header
2. The editor panel slides in from the right with sections for:
   - **Global Settings** — Change the base currency
   - **List Metadata** — Edit name, short name, category, tag, description
   - **Create New List** — Add a new watchlist with a unique slug
   - **Tickers** — Inline edit any ticker's symbol, name, tag, and currency; add or remove tickers
3. All changes are saved instantly to `data.db` via the API

---

## Maintenance Notes

- **Backup `data.db`** before running `migrate.py` — it destroys and recreates the database
- **Adding new currencies**: Just set the `currency` field on a ticker to any valid ISO 4217 code; the app will automatically fetch the FX pair
- **Data fetching**: All price data is fetched server-side via `yfinance`. If Yahoo Finance changes their API, update `yfinance` (`pip install --upgrade yfinance`)
- **Port**: Server runs on port 8000 by default. Change it in the `uvicorn.run()` call at the bottom of `server.py`
